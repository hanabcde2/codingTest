#### DFS, BFS  (그래프 탐색 알고리즘)
- DFS
  1. 하나를 몰아본다
  2. `재귀함수` 사용
  3. 수행 시간 관점에서는 복불복
- BFS
  1. 여러 개를 하나씩 본다
  2. `Queue`, `LinkedList` 사용
  3. 시간복잡도가 낮다

// 재귀함수가 가장 일반적

  3. 조합 유형 (모든 조합 만들기)
